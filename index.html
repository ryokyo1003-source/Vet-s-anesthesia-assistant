💡 Gemini APIキーの取得
  1. aistudio.google.com/apikey にアクセス
  2. Googleアカウントでログイン → 「APIキーを作成」
  3. 生成されたキー（AIzaSy...）を上に貼り付け

※ キーはこの端末のブラウザにのみ保存されます

💡 Claude APIキーの取得
  1. console.anthropic.com にアクセス
  2. アカウント作成/ログイン → Settings → API Keys
  3. 「Create Key」でキーを生成
  4. 生成されたキー（sk-ant-...）を上に貼り付け

モニター撮影時にAIがバイタル値を読み取ります（APIキー必須）
